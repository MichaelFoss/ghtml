# Privacy Policy

Last updated: July 18, 2026

GHTML is a Chrome extension that inserts user-provided HTML into Gmail
compose windows.

## Data GHTML Handles

GHTML handles the HTML text that you type or paste into its dialog. The
extension sanitizes that HTML in your browser before inserting it into
the Gmail compose window you selected.

GHTML also stores the following settings locally in Chrome extension
storage:

- the dialog's current HTML contents
- the dialog's saved position

This local storage is used only to restore the dialog between openings
and Gmail reloads.

## Data Collection

GHTML does not collect, transmit, sell, or share your data.

GHTML does not send your HTML, email drafts, browsing activity, Gmail
content, or extension settings to the developer or to any third party.

## Gmail Access

GHTML runs only on `https://mail.google.com/*` so it can add the HTML
insertion dialog to Gmail compose windows and insert sanitized HTML into
the selected compose editor.

## Permissions

GHTML requests the following Chrome permissions:

- `storage`: saves the dialog contents and position locally in your
  browser.
- `https://mail.google.com/*`: allows the extension to run inside Gmail,
  detect compose windows, and insert sanitized HTML.

## Data Retention

Saved dialog contents and position remain in local Chrome extension
storage until you clear extension data, uninstall the extension, or the
extension replaces that stored value.
